# Phase 01 Spec: Linear DS 완성

## 목표
Linear Data Structures(배열/연결리스트/스택/큐)의 모든 서브 챕터가 **교과서형 6섹션(Story/Features/Visualization/Complexity/Implementation/Practice)**을 갖추도록 정합성을 확보한다.

## 범위 (In Scope)
- Linear DS 하위 모든 `sub-concepts/*/config.ts`에서 필수 섹션 누락 여부 점검
- 누락 섹션 보완 (특히 Stack 계열)
- Phase 01 작업 결과를 문서로 기록

## 비범위 (Out of Scope)
- Non-Linear / Algorithms 콘텐츠 본문 완성
- UI/디자인 변경
- 인터랙티브 로직 개선

## 수용 기준 (Acceptance Criteria)
- Linear DS 모든 sub-concept config에 `story`, `features`, `complexity`, `implementation`, `practiceProblems` 포함
- 누락 섹션 0건
- 점검 결과가 `validation.md`에 기록됨

# CTP QA 체크리스트

## 1) 기본 실행
- [ ] `npm --prefix web run dev` 정상 실행
- [ ] `/insights/ctp/...` 진입 시 500/콘솔 에러 없음
- [ ] `?tocDebug=1` TOC 수집 정상

## 2) 시뮬레이션 공통
- [ ] 실행/정지/다음/이전/스크러버 동작
- [ ] 코드 수정 후 실행 시 시각화 반영
- [ ] 리셋 버튼 동작(시뮬레이션 초기화 + 재실행)
- [ ] 상태 패널 숨기기/보기 토글 동작
- [ ] 이벤트 로그가 상태 패널에 표시
- [ ] 콘솔 에러(무한 루프, Skulpt 에러) 없음

## 3) 그래프 계열
- [ ] 노드/간선이 깜빡이지 않고 유지 렌더링
- [ ] 간선 라벨에 `w=...` / `d=...` 표시
- [ ] `edge_relax` 시 간선 강조(색/두께/애니메이션)
- [ ] `node_finalize` 시 확정 애니메이션 표시
- [ ] 다각형 배치 적용(DFS/BFS/Dijkstra)
- [ ] 줌/드래그/리셋 동작

## 4) 정렬 계열
- [ ] 막대 높이가 데이터 값에 비례
- [ ] 비교/스왑 색상 가이드 표시
- [ ] 병합/힙 정렬 전용 플레이그라운드 정상

## 5) 자료구조 계열
- [ ] Queue/Deque/Hash 시뮬레이션 코드 실행 및 시각화 연동
- [ ] Hash 충돌/rehash/probe 경로 시각화 정상
- [ ] Trie/Union-Find 시각화 동작

## 6) 알고리즘 계열
- [ ] DFS/BFS: 방문/큐/스택/parent 흐름 확인
- [ ] 최단경로: dist 갱신이 간선 위에 표시
- [ ] 위상정렬: 진입차수 0 처리 흐름 확인
- [ ] MST: 선택 간선 강조 및 누적 표시

# 품질 고도화 계획

## 1) Content 고도화
- 정의/불변식/엣지케이스 문단 추가
- 평균/최악 복잡도 분리 표기
- 구현 코드: "직접 구현" + "표준 라이브러리" 병기
- 패턴/문제 연결 강화
 - Graph 챕터는 **구조/표현 중심**, DFS/BFS는 **개별 분리**로 간단 개념만 유지
 - Algorithms DFS/BFS는 **코딩테스트 대비형**으로 확장(패턴/유형 중심 목차 재구성)

## 2) Simulator 고도화
- 트랙 A (버튼 강화): 값 입력/상태 배지/시나리오 버튼
- 트랙 B (코드 기반): Skulpt + Adapter 기반 "코드 → 시각화"
- Hash 전용: 버킷/체이닝/프로빙/리해시 시각화
 - Graph 시각화 레이아웃은 **섹션 고정형**으로 운영(원형/계층형/허브형 분리)

## 3) TOC 정합성
- 정적 검증 유지
- 런타임 디버그(`?tocDebug=1`) 실행 기록
 - DFS/BFS 분리 반영(그래프/알고리즘 양쪽 TOC 업데이트)

# Quality Spec Pack

품질 관련 문서를 **Spec Kit** 방식으로 통합 관리합니다.

## Documents
- `spec.md`: 범위/정의/요구사항
- `analysis.md`: 비교 분석 + 점수 요약
- `plan.md`: 단계별 개선 계획
- `tasks.md`: 실행 체크리스트
- `policy.md`: 운영 정책(버튼/코드 모드)
- `mode-map.md`: 챕터별 모드 매핑
- `simulator-scores.md`: 점수 리포트
- `runtime-visualization-report.md`: 런타임 시각화 공백 이슈 리포트
- `custom-code-compat-strategy.md`: 커스텀 코드 시각화 호환 전략
- `content-depth-report.md`: 콘텐츠 분량/시뮬레이터 사용성 진단
- `qa-checklist.md`: 런타임/시뮬레이션 QA 체크리스트
- `qa-plan.md`: QA 수행 계획서

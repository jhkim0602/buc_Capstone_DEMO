# 시뮬레이터 모드 맵 (Button vs Code)

## 기준
- **Button**: 초기 개념 직관/상태 변화 강조
- **Code**: Skulpt 기반 "코드 → 시각화"로 구현 감각 강화

---

## Linear Data Structures

### Array
- 1D Array: **Code**
- 2D Array: **Code**
- String: **Code**

### Linked List
- Singly: **Code**
- Doubly: **Code**
- Circular: **Code**
- Two Pointers: **Code**

### Stack
- LIFO Basics: **Button**
- Array Stack: **Code**
- Linked Stack: **Code**
- Monotonic Stack: **Code**

### Queue
- Linear Queue: **Button** (Dead Space/Overflow 직관 강조)
- Circular Queue: **Code**
- Deque: **Code**
- Priority Queue Basics: **Code**

### Hash Table
- Hash Basics: **Button** (해시→버킷 매핑 직관)
- Collision Handling: **Code**
- Hash Implement: **Code**

---

## Non-Linear Data Structures

### Tree
- Tree Basics: **Button**
- Binary Traversal: **Code**
- BST: **Code**

### Heap
- Heap Basics: **Button**
- Min Heap: **Code**
- Max Heap: **Code**

### Graph
- Graph Representation: **Button**
- DFS/BFS: **Code**
- Cycle Detection: **Code**
- Shortest Path: **Code**
- MST: **Code**

### Trie
- Trie Basics: **Button**
- Prefix Search: **Code**
- Trie Applications: **Code**

### Union-Find
- Disjoint Set Basics: **Button**
- Union by Rank: **Code**
- Path Compression: **Code**
- Applications: **Code**

---

## Algorithms
- Sorting (Bubble/Selection/Insertion/Merge/Quick/Heap): **Code**
- Binary Search (Basic/Parametric): **Code**
- DFS (Basics/Backtracking/Tree Traversal/Cycle/Path): **Code**
- BFS (Basics/Grid/Multi-Source/0-1/Path): **Code**
- Shortest Path (Dijkstra/Floyd): **Code**
- Advanced Graph (Topo/MST): **Code**
- DP (Basics/1D/2D/Patterns): **Code**

1) Queue/Deque/Hash의 **Code 전환** (요청 우선순위)
